趁着年轻立稳根基

分享者:俊晔

一.但以理的见证

(但以理书 1:3-6 ) 王吩咐太监长亚施毗拿，从以色列人的宗室和贵胄中带进几个人来，就是年少没有残疾、相貌俊美、通达各样学问、知识聪明俱备、足能侍立在王宫里的，要教他们迦勒底的文字言语。王派定将自己所用的膳和所饮的酒，每日赐他们一份，养他们三年。满了三年，好叫他们在王面前侍立。他们中间有犹大族的人：但以理、哈拿尼雅、米沙利、亚撒利雅。

但以理年少被俘，却因着立下稳固的根基，在数不清的逼迫中得胜。

二.年轻的圣洁

(提摩太后书 2:21 ) 人若自洁，脱离卑贱的事，就必作贵重的器皿，成为圣洁，合乎主用，预备行各样的善事。

约瑟注重身体的圣洁，因对神的敬虔，抵挡住了波提乏妻子的诱惑，即便因此遭遇逼迫。他在监牢里会后悔当初拒绝波提乏妻子吗？必然是不会的，正是他的自洁，脱离卑贱的事，才会被神预备，才会有将来成为合乎主用的器皿的光景。

三.给我们的警示

年轻的弟兄姊妹，来到团契，不是要干别的事情，是要在神家里有学习、有操练，回到本家能够去服事教会。大学是自由的，我们绝不能被情欲、世界的诱惑裹挟走，失去宝贵的圣洁。应该把握住时间，利用好这些光阴，在神的话语上有更多的装备、有更多的追求，趁年轻立稳根基。

我与主的关系

分享者:恩超

一.主对我们的爱

(约翰福音 3:16 ) “　神爱世人，甚至将他的独生子赐给他们，叫一切信他的，不至灭亡，反得永生。

这是每一个基督徒都无比熟悉的一节经文，神对我们无私、宽宏、广大的爱显露无遗。即便基督已经升天，即便我们的服事不可能如彼得、保罗一样写入圣经，但是主实在是看见我们的困苦，实在是爱着我们。

　　(以弗所书 3:20 ) 神能照着运行在我们心里的大力充充足足地成就一切，超过我们所求所想的。

无论我们在生活中遇到什么样的忧愁，只要一开始便将难处接着祷告求告主，主便会为我们成就一切，赐给我们的恩，必会超过我们所求所想。

二.我们去爱主

(出埃及记 38:7-8 ) 把杠穿在坛两旁的环子内，用以抬坛，并用板做坛；坛是空的。他用铜做洗濯盆和盆座，是用会幕门前伺候的妇人之镜子做的。

坛是空的，总要有人去献上自己。在这个末了的时代，我们需要人去献上祭品。我们愿意将自己完全摆上，主为我们牺牲，我们也愿献上自己跟随主。

镜子是妇人最宝贵的东西，却甘心献给主用作会幕门前的伺候。我们要问自己，能不能像献上镜子的妇人一样，将最心爱的东西献给主，将自己交给主。

三.多祷告、多见证

圣经有关祷告的经文数都数不过来，祷告实在是亲近主的桥梁、是向主认亏欠的桥梁、是我们去听主话语的桥梁、是充满信心的桥梁、是荣耀赞美神的方式。祷告的重要性不言而喻，弟兄姊妹实在要常常祷告，与神交通。祷告是一件简单的事情，我们不必有华丽的词藻、不必有太多的修辞，只要发自内心地亲近主，哪怕只有“主啊，我爱你”，也足够了。

我们年轻一代虽然没有属灵前辈们那般勇敢，虽然还没有得到神的重用，但是这不代表我们就没有足够的见证。生活中，不信的朋友、学习、世界的诱惑都是试炼，都是考验，而神需要我们做的，就是在各种事项上为主做美好的见证。

根基的问题

分享者:盛溢

(马太福音 7:25 ) 雨淋，水冲，风吹，撞着那房子，房子总不倒塌，因为根基立在磐石上。

(路加福音 6:46-49 ) “你们为什么称呼我‘主啊，主啊’却不遵我的话行呢？凡到我这里来，听见我的话就去行的，我要告诉你们他像什么人：他像一个人盖房子，深深地挖地，把根基安在磐石上；到发大水的时候，水冲那房子，房子总不能摇动，因为根基立在磐石上（有古卷：因为盖造得好）。惟有听见不去行的，就像一个人在土地上盖房子，没有根基；水一冲，随即倒塌了，并且那房子坏的很大。”

(哥林多前书 3:10 )我照　神所给我的恩，好像一个聪明的工头，立好了根基，有别人在上面建造；只是各人要谨慎怎样在上面建造。

(诗篇 11:3 ) 根基若毁坏，/义人还能做什么呢？

我们怎么样才算得着、才算改变、才算是得着了新的亮光呢？

我们这一条道路，不是知识的道路、道理的道路；也不是一群人情感上简单地碰撞一下，千百次聚会，只为遇见主。我们遇见了主，当场我们心里就改变过来，回去后有心志、有方向、有路可走，这才是真正地在聚会里遇见了主。

一.根基的重要

一栋房子，若是不打地基，越建造它就越容易倒塌、亏损得就越大。打下根基的时候，我们看不出来，但是后来的工程就让根基的重要性显现出来。主耶稣告诉我们，建在磐石上的房子和建在沙土上的房子，当同样的环境临到时，显示出两种完全不一样的结果。圣经告诉我们，“凡听见我的话，就去行的”，听见主的话再去行，根基是非常牢固的。

耶稣是永活的神，祂一直在与我们说话，但是我们总是捂着耳朵听不见；我们就是听见了主的话，也总是犹豫去行还是不去行。保罗已经告诉我们，这个根基就是主耶稣。连世人都知道，你的思路决定了你的出路。我们基督徒在主面前所行的，也决定了我们以后属灵在神面前的结果。

保罗说我们不必在意根基，因为主耶稣就是我们的根基。我们以基督为出发点，那么我们所行的，跟基督就是分不开的。

神国的权能不在乎言语、不在乎知识，在乎的是圣灵的权能、是圣灵的平安。不是知识、体系成为在主面前立下根基的基础性的东西，而是基督在你里头，你在基督里头的经历，成为实打实的生命的根基。

二. 如何打根基

从创世纪开始，那么多古圣先贤，他们在神面前以主为重的生活给我们做了美好的榜样。约瑟万不敢做这大恶得罪神。把神放在心里，这不是知识的劝导，而是跟神之间有基础。为什么以马忤斯的门徒要强留他们认不出来的主耶稣？只是因为与主在一起的感觉非常舒服，好像一离开就有失落感，所以强留是有基础的。

从主那里领受，我们知道去爱怎样的人，知道怎样去爱人，也知道自己的内心是怎样的。弟兄总说，趁着年轻立稳根基，趁着年轻多来到主的面前，基督成为我们的根基。
